CSS

CSS also known as Cascading style sheet.This is used to make dynamic page using colours,styling,padding,alligning etc.
It can be used as inline,internal,external

  • Inline CSS

Inline CSS is used when we want use it as different line and accordingly we wnat to make changes in that particular html tag.
Inline CSS used when we make small projects and on individual line.

  • Internal CSS

Internal CSS is used when we wnat to make changes on many attributes of same html tag and therefore we make changes by introducing them on head inside style attribute.
Here we intoduce them with html tag inside curly brackets and make changes like padding,size,color,margin etc.
